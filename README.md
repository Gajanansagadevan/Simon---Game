# 🎮 Simon Game

A modern browser-based **Simon Memory Game** built using **HTML, CSS, and JavaScript**. Test your memory by repeating an increasingly longer sequence of colors. As the game progresses, each level becomes more challenging.

## 🌐 Live Demo

Play the game here:

https://gajanansagadevan.github.io/Simon---Game/

---

## 📌 Features

- 🎨 Modern and responsive UI
- 🎵 Sound effects for each color
- 🧠 Increasing difficulty with every level
- 🏆 High score saved using Local Storage
- ✨ Smooth animations and visual effects
- 📱 Mobile-friendly design
- ⌨️ Keyboard support
- 🔄 Restart game anytime

---

## 🕹️ How to Play

1. Click the **Start** button.
2. Watch the sequence of flashing colors.
3. Repeat the sequence by clicking the colored buttons.
4. Each correct round adds one more color to the sequence.
5. If you click the wrong color, the game ends.
6. Try to beat your highest score!
